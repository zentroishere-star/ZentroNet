## 🚀 QUICK START

### Windows (Automatic)
```bash
setup.bat
```
→ Creates venv, installs packages, runs migrations, creates admin account, starts server

### Mac/Linux (Automatic)
```bash
chmod +x setup.sh
./setup.sh
```
→ Same as above

### Manual
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

### Access
- Website: http://127.0.0.1:8000/
- Admin: http://127.0.0.1:8000/admin/
